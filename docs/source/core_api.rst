.. currentmodule:: pyinterp

Core API Documentation
######################

Low-level classes and functions implemented in the compiled extension module
(``pyinterp.core``). These provide direct access to the C++ implementations
and offer fine-grained control over data types and algorithms.

Most users should start with the high-level :doc:`api` overview, which provides
a more convenient interface with automatic type handling.
